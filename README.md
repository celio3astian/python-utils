# Python Utilities

Pequenos utilitários em Python: calculadora, conversor de temperatura e manipulação básica de listas.

## Arquivos
- [main.py](main.py) — implementa as funcionalidades e o menu principal.


## Funcionalidades
- Calculadora simples: [`main.calculadora`](main.py)
- Conversor de temperatura (°C → °F / K): [`main.conversor_temperatura`](main.py)
- Manipulação de dados (lista, média, maior, menor): [`main.manipulacao_dados`](main.py)
- Menu e fluxo principal: [`main.menu`](main.py)

## Requisitos
- Python 3.x

## Execução
Abra um terminal e execute:
```sh
python main.py
```

## Uso
Siga as instruções interativas no menu para escolher a funcionalidade desejada.
